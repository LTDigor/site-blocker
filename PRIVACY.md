# Site Blocker Privacy Policy

Site Blocker does not collect, sell, share, or transmit user data to the developer or any third party.

The extension stores its block list, rule settings, temporary unblock state, and optional custom block image locally in Chrome extension storage on the user's device. This data is used only to provide the extension's website-blocking functionality.

Site Blocker does not use remote code, analytics, tracking, advertising, or external servers.
