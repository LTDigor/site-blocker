# Chrome Web Store assets

Store listing assets live here so they are versioned with the release but excluded from the extension ZIP.

Replace this file with final assets before the first manual store submission:

- At least one screenshot for the Chrome Web Store listing.
- Promo tile: `440x280`.
- Optional larger promo images if you want to use them in the listing.

The extension icon used by Chrome itself lives in `assets/icons/` and is included in the package.
